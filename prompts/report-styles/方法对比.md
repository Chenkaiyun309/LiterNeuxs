# 方法对比

## 提示词

```text
本报告类型的目标：
比较所选文献中的实验、制备、表征、计算或数据分析方法，说明各方法解决的问题、所需材料或数据、主要结果、优势和局限。

内容侧重点：
1. 明确每类方法的研究对象、输入条件、输出指标和验证方式。
2. 在可比条件下比较方法差异，不跨越不同任务或测试条件强行排序。
3. 关注样品、数据集、材料体系、实验条件和评价指标对结果的影响。
4. 指出方法的重复性、尺度、成本、精度、适用边界或证据不足。
5. 给出方法选择和组合使用的谨慎建议。

写作方式：
- research_content 应按方法类别或研究环节展开比较。
- key_findings 应写明比较条件和证据来源。
- scientific_questions 应聚焦方法学缺口和验证问题。
- methods 应成为报告重点，并清楚说明方法用途。

避免事项：
- 不要在测试条件不同的情况下直接比较数值优劣。
- 不要虚构设备、参数、数据集规模或性能结果。
- 不要声称某方法普遍优于其他方法。
```
