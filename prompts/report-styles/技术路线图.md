# 技术路线图

## 提示词

```text
本报告类型的目标：
梳理本批文献中“材料体系-制备/处理工艺-结构表征-性能目标-应用场景”之间的路径关系，帮助研究人员识别可延伸的技术路线和关键控制节点。

内容侧重点：
1. 明确不同材料体系或研究对象分别对应哪些工艺路线、改性策略或性能目标。
2. 提炼制备方法、后处理方式、成分设计、微结构调控、表征手段和性能评价之间的因果链条。
3. 比较不同技术路线的优势、限制、适用场景和可迁移性。
4. 识别路线中的关键变量，例如成分、组织结构、界面、缺陷、热处理、载荷环境、测试条件或数据分析方法。
5. 指出可以继续延伸的路线，但必须以输入文献已经出现的材料、方法或问题为依据。

写作方式：
- research_content 应像技术路线解析，而不是一般综述。
- key_findings 应突出“哪条路线解决了什么问题、还受什么限制”。
- scientific_questions 应偏向路线瓶颈、变量耦合、结构-性能映射、尺度放大和稳定性问题。
- methods 应尽量按技术链条表达，例如“成分设计-制备工艺-表征验证-性能评价”。

避免事项：
- 不要只总结研究结论而忽略方法路径。
- 不要凭空补充未在文献摘要中出现的工艺参数、设备条件或性能数值。
- 不要把不同适用场景的技术路线简单排序为优劣。
```
