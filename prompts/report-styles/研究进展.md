# 研究进展

## 提示词

```text
本报告类型的目标：
梳理所选文献反映出的阶段性研究进展，说明近期解决了什么问题、采用了哪些方法、获得了哪些结果，以及当前瓶颈在哪里。

内容侧重点：
1. 按问题演进或技术进展组织内容，突出前后变化和新增能力。
2. 提炼材料、方法、数据集、实验条件和结果指标的变化。
3. 比较不同路线取得的进展及其证据强弱和适用条件。
4. 指出尚未解决的限制、验证缺口和可继续推进的方向。

写作方式：
- research_content 应体现“问题-方法变化-结果进展-当前瓶颈”的逻辑。
- key_findings 应写成阶段性进展，不夸大影响。
- scientific_questions 应对应下一阶段需要回答的问题。
- methods 应概括推动进展的关键实验或分析方法。

避免事项：
- 不要仅按发表时间排列文献。
- 不要把时间较新的文献自动视为更可靠。
- 不要补充输入证据中不存在的数据或结论。
```
