# 专利机会

## 提示词

```text
本报告类型的目标：
从本批文献中识别潜在可保护的技术组合、应用场景、差异化改进点和后续专利挖掘方向，为专利构思和技术交底提供前期线索。

内容侧重点：
1. 优先关注材料组成、结构设计、制备工艺、处理流程、表征/评价方法和应用场景之间的组合关系。
2. 提炼可能形成保护点的技术特征组合，例如“材料体系 + 工艺步骤 + 目标性能 + 应用环境”。
3. 识别与已有路线相比的差异化改进点，但只能基于输入文献中的描述进行归纳。
4. 关注潜在规避空间，例如可替换材料、可变工艺路线、可扩展应用场景或可调整评价指标。
5. 区分“文献已明确支持的事实”和“值得进一步验证的专利机会”。

写作方式：
- research_content 应突出技术机会图谱，而不是普通综述。
- key_findings 可写成“潜在技术机会”或“可保护技术组合”的阶段性判断。
- scientific_questions 应偏向待验证保护点、效果支撑不足、实施边界、可替代方案和侵权规避风险。
- methods 应概括形成专利方案所依赖的材料设计、工艺路径、验证方法和应用测试。

避免事项：
- 不要直接声称某方案具备新颖性、创造性或可授权性。
- 不要虚构实验效果、性能提升幅度、应用结果或产业价值。
- 不要生成权利要求书式绝对表述；这里只做专利机会挖掘和交底前线索整理。
```
