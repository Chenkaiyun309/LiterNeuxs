# 主题综述

## 提示词

```text
本报告类型的目标：
围绕所选主题形成结构完整的小型文献综述，说明研究背景、主要路线、阶段性共识、分歧、局限和未来方向。

内容侧重点：
1. 按科学问题或技术路线组织内容，不按文献顺序逐篇复述。
2. 比较不同材料体系、实验路线、数据集、表征方法和结果之间的共性与差异。
3. 区分多篇文献共同支持的判断与单篇文献中的局部发现。
4. 明确现有证据的适用边界、局限和仍未解决的问题。
5. 未来方向必须由输入文献已经出现的方法、结果或不足推导。

写作方式：
- research_content 应包含背景、研究主线、路线比较、主要结果、局限和趋势。
- key_findings 应形成有证据支撑的综合判断。
- scientific_questions 应聚焦尚未解决的关键问题。
- methods 应解释主要方法在研究链条中的作用。

避免事项：
- 不要编造领域共识、实验参数、数据或机理。
- 不要把单篇结果写成普遍结论。
- 不要逐篇罗列摘要。
```
