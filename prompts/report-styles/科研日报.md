# 科研日报

## 提示词

```text
本报告类型的目标：
面向日常快速跟踪，帮助研究人员在较短时间内把握本批文献中的最新进展、代表性发现、关键问题和后续关注点。

内容侧重点：
1. 优先提炼近期出现的新材料体系、新工艺路线、新表征方法、新性能目标或新应用场景。
2. 强调“哪些进展值得今天关注”，而不是写成完整领域综述。
3. 对不同文献之间的共同趋势和差异做简洁归纳，避免逐篇摘要式复述。
4. 对研究问题、方法变化和潜在影响进行克制判断，判断必须来自输入文献。

写作方式：
- 语言应简洁、清晰、学术化，适合科研日报或课题组内部动态通报。
- research_content 应突出“最新动态 + 代表性方法 + 当前问题 + 后续趋势”的逻辑。
- key_findings 应写成值得关注的阶段性进展或趋势判断。
- scientific_questions 应聚焦本批文献暴露出的未解决问题。
- methods 应概括本批文献中反复出现或有代表性的技术路线。

避免事项：
- 不要写成过长的百科式综述。
- 不要夸大单篇文献影响。
- 不要引入输入文献之外的背景、数据、机理或应用前景。
```
