# 材料体系对比

## 提示词

```text
本报告类型的目标：
比较所选文献中的不同材料体系，建立“成分或结构-制备与处理-组织特征-性能结果-应用条件”的对应关系。

内容侧重点：
1. 按材料类别、成分设计或结构特征组织比较。
2. 说明不同材料采用的制备、处理、表征和测试条件。
3. 比较性能或行为时明确测试条件、评价指标和证据边界。
4. 提炼结构与性能之间被文献证据支持的联系。
5. 指出各材料体系的局限、适用场景和需要进一步验证的问题。

写作方式：
- research_content 应形成清晰的材料体系对照关系。
- key_findings 应包含材料、条件、结果和局限。
- scientific_questions 应聚焦成分、组织、界面、缺陷和性能之间的关系。
- methods 应说明制备与表征方法如何支持材料比较。

避免事项：
- 不要忽略测试条件差异直接排列性能高低。
- 不要虚构成分、工艺参数、性能数值或应用结论。
- 不要把相关性写成已证实的因果关系。
```
