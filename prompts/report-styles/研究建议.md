# 研究建议

## 提示词

```text
本报告类型的目标：
基于本批文献中已经出现的材料、方法、问题和不足，提炼下一步可执行的研究假设、变量设计、对照方案、表征手段和风险点。

内容侧重点：
1. 将文献中的研究不足转化为可验证的研究问题或研究假设。
2. 明确建议关注的变量，例如成分、工艺条件、处理路径、微结构特征、服役环境、测试指标或数据分析方法。
3. 给出对照思路，例如基准材料、不同工艺组、不同处理条件、不同测试环境或不同表征尺度。
4. 说明推荐表征或测试方法的作用，例如验证相组成、组织演化、界面状态、力学性能、腐蚀行为、稳定性或失效机制。
5. 提醒研究风险和不确定性，例如摘要信息不足、变量耦合、测试条件差异、样品制备可重复性或结论外推风险。

写作方式：
- research_content 应突出“文献依据-研究假设-变量设计-验证路径-风险控制”的逻辑。
- key_findings 应写成能够支撑研究设计的依据或观察。
- scientific_questions 应转化为可检验的问题，而不是宽泛的研究空白。
- methods 应尽量具体说明“方法用于验证什么”，但不得编造文献未提供的参数数值。

避免事项：
- 不要提出与输入文献完全无关的研究路线。
- 不要虚构具体实验参数、性能数值或机理结论。
- 不要把探索性建议写成已经被证明的结论。
```
