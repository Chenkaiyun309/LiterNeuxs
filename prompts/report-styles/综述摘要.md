# 综述摘要

## 提示词

```text
本报告类型的目标：
形成一份可用于阶段性汇报、开题背景、研究进展整理的小型综述摘要，重点帮助读者理解该方向的研究主线、已有共识、路线分歧和关键瓶颈。

内容侧重点：
1. 首先交代研究背景和问题来源，说明该方向为什么值得研究。
2. 提炼多篇文献之间的共同研究主线，而不是按文献顺序罗列。
3. 比较不同材料体系、工艺路线、表征方法或理论分析框架之间的共性与差异。
4. 概括已有工作形成的阶段性共识，并指出仍然存在的不确定性、争议或不足。
5. 对未来趋势做谨慎归纳，趋势判断必须基于输入文献中已经出现的对象、方法或问题。

写作方式：
- 语言应接近正式“研究进展综述”或“领域动态综述”。
- research_content 应最完整，重点展开背景、主线、方法比较、瓶颈和趋势。
- key_findings 应写成具有综述感的阶段性结论，而不是新闻标题。
- scientific_questions 应偏向基础科学问题、结构-性能关系、稳定性、机制解释和方法学限制。
- methods 应说明不同方法在研究链条中的作用。

避免事项：
- 不要写成日报式短讯。
- 不要逐篇介绍“某研究做了什么”。
- 不要把尚未形成共识的单篇结果表述为领域共识。
```
